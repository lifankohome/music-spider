# Music-spider 橡皮音乐

### Introduction

在线音乐播放器，歌曲库：网易 + 酷狗 + QQ + 虾米 + 百度，合称中华曲库

### Feature

 + HTML5 Player
 + Lyric Display
 + 180px Album Display
 + Hot Search Word Display
 + Lyric Spacing Auto Adjust
